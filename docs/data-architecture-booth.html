7RAG Indexing Pipeline

  • Publish/update enqueues re-embed of changed chunks only (hash diff).
  • Embedding model + dim stored as metadata → deterministic re-index on model change.
  • Query path: embed → cosine kNN over published chunks within validity window.

12Security & PDPA (Thailand)

  • Consent: purpose-based, versioned, captured at LINE onboarding.
  • Minimization: pseudonymous line_user_id; no phone/national ID.
  • Webhook integrity: HMAC-SHA256 signature verify + idempotency dedupe.
  • Erasure/portability: cascade delete + time-limited export; audit retained lawfully.
  • RBAC: viewer/editor/dpo; PII view logged to append-only audit.
  • Residency: in-region stores + LLM egress PII-redacted; DPA with processors.
  • Encryption: TLS in transit; at-rest DB encryption; secrets in env vault.
